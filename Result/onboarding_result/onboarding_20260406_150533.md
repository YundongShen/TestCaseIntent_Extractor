# TestSubject

## TestObjectives

* Verify the visibility of the "GET IN TOUCH" form when the user enters their name, email, subject, and message, and uploads a file.
* Verify that the user clicks the "Submit" button and the "OK" button.
* Verify that the "successMessage" is displayed after the user clicks the "OK" button.
* Verify that the "Home" button is visible after the user completes the form.

## TestActivitySteps

1. Open the application and navigate to the "Home" page.
2. Click on the "GET IN TOUCH" button to display the form.
3. Enter a valid name, email, subject, and message in the respective fields.
4. Upload a file by selecting it from your device.
5. Click the "Submit" button to send the form data.
6. Click the "OK" button to acknowledge the submission.
7. Verify that the "successMessage" is displayed after clicking the "OK" button.
8. Verify that the "Home" button is visible after completing the form.

## KeyCodeSnippets

* To verify the visibility of the "GET IN TOUCH" form, use the following code snippet:
```javascript
cy.get('.contact-us-form').should('be.visible');
```
* To verify that the user clicks the "Submit" button, use the following code snippet:
```javascript
cy.get('.submit-button').click();
```
* To verify that the user clicks the "OK" button, use the following code snippet:
```javascript
cy.get('.ok-button').click();
```
* To verify that the "successMessage" is displayed, use the following code snippet:
```javascript
cy.get('.success-message').should('be.visible');
```
* To verify that the "Home" button is visible, use the following code snippet:
```javascript
cy.get('.home-button').should('be.visible');
```

## CoreConcepts

* The "GET IN TOUCH" form is used to contact the support team.
* The form requires the user to enter their name, email, subject, and message, and upload a file.
* The user must click the "Submit" button to send the form data, and then click the "OK" button to acknowledge the submission.
* After completing the form, the "Home" button is displayed, allowing the user to return to the home page.
