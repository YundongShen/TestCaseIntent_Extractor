# Test Subject

# Test Objectives

* Verify the visibility of the "Get in touch" button on the contact us page
* Verify that the user can submit the contact form with details and that the details are submitted successfully
* Verify that the user can upload a file
* Verify that the user can click the submit button
* Verify that the user can click the OK button
* Verify that a success message "Your details have been submitted successfully" is visible
* Verify that the user can click on the home button
* Verify that the user lands on the home page successfully

# Test Activity Steps

### Step 1: Open the contact us page

* Verify that the page is loaded successfully

### Step 2: Verify the visibility of the "Get in touch" button

* The "Get in touch" button should be visible on the page

### Step 3: Enter the user details

* The user should enter their name, email, subject, and message in the respective fields

### Step 4: Upload a file

* The user should be able to upload a file from their device

### Step 5: Click the submit button

* The user should click the submit button to send the contact form

### Step 6: Click the OK button

* The user should click the OK button to confirm the submission

### Step 7: Verify the success message

* A success message "Your details have been submitted successfully" should be visible on the page

### Step 8: Click on the home button

* The user should click on the home button to navigate back to the home page

### Step 9: Verify the landing on the home page

* The user should successfully land on the home page after clicking on the home button
