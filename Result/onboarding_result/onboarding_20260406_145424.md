# Test Subject

Test Subject: Connecting to Test-API.k6.io

## Test Objectives

* Verify that 10 virtual users can connect to the test API.k6.io
* Check that the HTTP GET request returns a 200 status code and response time is acceptable (less than 500ms)

## Test Activity Steps

1. Open a web browser and navigate to 'https://test-api.k6.io'
2. Perform an HTTP GET request to the URL
3. Verify that the response status code is 200
4. Check that the response time is less than 500ms
5. Sleep for 1 second

## Key Code Snippets

```javascript
// Perform HTTP GET request
const response = await fetch('https://test-api.k6.io');

// Check response status code
if (response.status === 200) {
  console.log('Response status code is 200');
} else {
  console.log('Response status code is not 200');
}

// Check response time
if (response.timings.duration < 500) {
  console.log('Response time is acceptable');
} else {
  console.log('Response time is not acceptable');
}

// Sleep for 1 second
sleep(1);
```
